# Quick Start: Test Your PWA

## 3 Simple Steps:

### 1️⃣ Build
```bash
npm run build
```

### 2️⃣ Preview
```bash
npm run preview
```

### 3️⃣ Install
Open the URL shown (usually http://localhost:4173) in your browser:

**Desktop:** Look for install icon in address bar OR click browser menu (⋮) → "Install GymFlow"

**Mobile:** Browser menu → "Install app" or "Add to Home Screen"

**In-App:** Go to Settings page → Click "Install App" button

---

## That's it! 🎉

The app will:
- Open in its own window
- Work offline
- Appear on your home screen/app drawer

For detailed instructions, see `HOW-TO-TEST-PWA.md`
